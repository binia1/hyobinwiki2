        <h2 id="s-1">1. 개요</h2>
        <div class="quote-box">
            "오만한 독재자를 바다에 던져버리고, '신세계(New World)'로 출항하다."<br>
            "<a href="칠양여객.html" class="wiki-link">칠양여객</a>, <a href="안천고속.html" class="wiki-link">안천고속</a>과 함께 효빈시 고속버스 3대장을 지키는 '자유의 바람'."
        </div>
        <p><a href="효빈광역시.html" class="wiki-link">효빈광역시</a> <a href="동구(효빈광역시).html" class="wiki-link">동구</a> <a href="전천동.html" class="wiki-link">전천동</a>을 거점으로 하는 시외/고속버스 운송 업체. <a href="칠양여객.html" class="wiki-link">칠양여객</a> 다음가는 규모를 자랑하는 거대 운송사지만, 2015년 이전까지는 이름값 못하는 <strong>'쓰레기 회사'</strong>의 대명사였다.</p>
        <p class="mt-2">과거 사장 <strong><a href="오만한.html" class="wiki-link">오만한</a></strong>(실명)은 <a href="윤대환.html" class="wiki-link">윤대환</a>에게 빌붙어 직원들을 착취하고, 수익성 없는 단거리 노선 알박기로 보조금만 타 먹던 기생충 같은 경영자였다. 그러나 2015년 <a href="김성민.html" class="wiki-link">김성민</a> 시장의 서슬 퍼런 '면허 몰수' 협박에 직원들이 봉기하여 오만한을 축출하는 <b>'덕빈의 난'</b>을 일으켰다.</p>
        <p class="mt-2">이후 새롭게 취임한 <a href="신세계.html" class="wiki-link">신세계</a> 대표의 지휘 아래, 돈 안 되는 단거리 노선을 과감히 정리하고 광역버스와 초장거리 고속 노선에 올인하는 '선택과 집중' 전략으로 화려하게 부활했다.</p>

        <h2 id="s-2">2. 역사: "대해적시대의 개막"</h2>
        
        <h3 id="s-2-1">2.1. 오만한의 폭정 (1998~2015)</h3>
        <p>전임 사장 <a href="오만한.html" class="wiki-link">오만한</a>은 이름 그대로 오만방자했다. <a href="두청운수.html" class="wiki-link">두청운수</a>가 망한 뒤에도 정신을 못 차리고, 직원들에게 폭언을 일삼으며 회사 돈으로 도박을 했다. 시외버스 면허를 가지고 시내버스처럼 단거리 구간만 뺑뺑이 돌리며 보조금을 타 먹는 꼼수 경영을 지속하여 시민들의 비웃음을 샀다.</p>

        <h3 id="s-2-2">2.2. 김성민의 처형식과 반란 (2015)</h3>
        <p>2015년, 교통 개혁을 추진하던 <strong><a href="김성민.html" class="wiki-link">김성민</a></strong> 시장이 오만한을 소환해 "다음 달까지 정상화 안 하면 공중분해 시키겠다"는 일명 <b>'버스터 콜(Buster Call)'</b>을 발동했다. 회사가 없어질 위기에 처하자 노조와 직원들이 들고일어났다. 그들은 사장실 문을 부수고 들어가 오만한을 의자째로 들어서 회사 밖으로 내동댕이치고, 횡령 혐의로 경찰에 넘겨버렸다.</p>

        <h3 id="s-2-3">2.3. 신세계로의 출항 (2016~현재)</h3>
        <p>노조위원장 출신의 <a href="신세계.html" class="wiki-link">신세계</a>가 신임 대표로 선출되었다. 그는 《<a href="원피스(만화).html" class="wiki-link">원피스</a>》 광팬으로, 취임사에서 <b>"나는 버스왕이 될 사나이다!"</b>를 외쳤다. 이후 단거리 노선을 정리하고 수도권행 <strong>광역급행버스(M버스)</strong>와 땅끝마을까지 가는 <strong>초장거리 노선(Grand Line)</strong>을 대거 신설해 업계 2위권의 강자로 도약했다.</p>

        <h2 id="s-3">3. CEO 신세계: "캡틴(Captain)"</h2>
        <p>40대 후반. 덥수룩한 수염에 중절모를 즐겨 쓰며, 호탕한 성격으로 직원들을 <b>"동료(Nakama)"</b><span class="wiki-fn">2<span class="wiki-tooltip">동료라고 쓰고 '나카마'라고 읽는다.</span></span>라고 부른다.</p>
        <ul class="list-disc pl-5 mt-2">
            <li><strong>최애(Oshi):</strong> <a href="몽키_D._루피.html" class="wiki-link">몽키 D. 루피</a>, <a href="흰_수염.html" class="wiki-link">흰 수염</a>.</li>
            <li><strong>경영 철학:</strong> "내 배(버스)에 탄 녀석들은 모두 내 가족이다."</li>
            <li><strong>특이사항:</strong> 과거 핍박받던 기억 때문에 <b>"직원이 굶으면 사장은 할복해야 한다"</b>는 신념을 가지고 있다. 회식 때 술통을 통째로 들고 건배 제의를 한다.</li>
        </ul>

        <h2 id="s-4">4. 특징: "그랜드 라인(Grand Line)"</h2>
        <p>덕빈고속은 장거리 여행의 설렘과 모험심을 자극하는 컨셉을 잡았다. 본사가 위치한 <a href="전천동.html" class="wiki-link">전천동</a>은 과거 거친 분위기의 지역이었으나, 지금은 덕빈고속의 '모험 기지' 역할을 하고 있다.</p>

        <h3 id="s-4-1">4.1. 써니호 플릿 (Sunny Fleet)</h3>
        <ul class="list-disc pl-5 mt-2 text-sm">
            <li><strong>차량:</strong> 프리미엄 골드 버스에 '고잉 메리호', '싸우전드 써니호' 등의 애칭을 붙였다. 좌석마다 귀여운 해골 마크 쿠션이 있다.</li>
            <li><strong>도색:</strong> 밝은 <strong>주황색(오렌지)</strong>과 노란색을 섞은 활기찬 도색을 사용하여 멀리서도 눈에 확 띈다.</li>
        </ul>

        <h3 id="s-4-2">4.2. 장거리/광역 특화</h3>
        <p>해남, 완도, 속초 등 4시간 이상 소요되는 <b>'그랜드 라인(초장거리)'</b> 노선을 독점하다시피 한다. 또한 <a href="안천고속.html" class="wiki-link">안천고속</a>(붉은 혜성)과 연합하여 효빈시 광역버스망의 양 날개를 담당한다.</p>

        <h3 id="s-4-3">4.3. 사내 문화</h3>
        <p>수직적인 문화를 철폐했다. 사장도 기사도 서로 "어이, 김 선장!", "박 항해사!"라고 부르며 농담을 주고받는다. 장거리 운전 기사들을 위해 터미널 휴게실에 최고급 수면실과 사우나를 완비했다.</p>

        <h2 id="s-5">5. 타 운송사와의 관계</h2>
        <ul class="list-disc pl-5 mt-2">
            <li><strong>vs <a href="칠양여객.html" class="wiki-link">칠양여객</a> (<a href="옥덕호.html" class="wiki-link">옥덕호</a>):</strong> <strong>[사황 간의 대립]</strong>. 업계 1위 칠양여객과는 선의의 경쟁자다. 옥덕호가 아이돌을 논하면 신세계는 "버스는 사나이의 모험심을 자극해야지!"라고 받아친다.</li>
            <li><strong>vs <a href="안천고속.html" class="wiki-link">안천고속</a> (<a href="안전호.html" class="wiki-link">안전호</a>):</strong> <strong>[동맹]</strong>. 같은 고속/광역 주력이라 죽이 잘 맞는다. 안천이 3배 빠르다고 자랑하면 덕빈은 "기어 세컨드!"를 외친다.</li>
            <li><strong>vs <a href="오만한.html" class="wiki-link">오만한</a> (전 사장):</strong> <strong>[현상수배범]</strong>. 오만한이 회사 근처를 기웃거리면 직원들이 소금을 뿌려 쫓아낸다. 신세계 대표는 "내 눈에 띄면 고무고무 총난타를 날려주겠다"고 경고했다.</li>
        </ul>

        <h2 id="s-6">6. 사건 사고</h2>
        <h3 id="s-6-1">6.1. 윤재훈 오물 투척 미수 사건 (2023)</h3>
        <p>덕빈고속은 이 사건을 <b>'해적의 명예'</b>를 걸고 비판했다. 신세계 대표는 SNS에 <strong>[현상수배: <a href="윤재훈.html" class="wiki-link">윤재훈</a> / 현상금 10베리]</strong> 포스터를 합성해 올리며 "남의 배에 똥을 던지는 건 해적도 안 하는 짓이다. 그건 양아치나 하는 짓이지."라고 일갈해 네티즌들의 환호를 받았다.</p>
